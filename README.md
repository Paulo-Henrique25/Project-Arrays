# 🍔 Dev Burguer

Projeto de uma aplicação web para pedidos de hambúrguer, desenvolvido com foco em prática de Front-End e criação de interfaces modernas e responsivas.

---

## 📌 Sobre o Projeto

O **Dev Burguer** é um projeto que simula uma hamburgueria online, onde o usuário pode visualizar produtos, selecionar opções e interagir com a interface de forma simples e intuitiva.

Este projeto foi desenvolvido com o objetivo de aprimorar habilidades em desenvolvimento web, aplicando conceitos importantes de JavaScript na prática.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Flexbox / Responsividade  

---

## 🧠 Conceitos Aplicados

Neste projeto foram utilizados métodos importantes de manipulação de arrays em JavaScript:

- 🔁 `forEach` → para percorrer e exibir os itens  
- 🗺️ `map` → para transformar dados  
- 🔎 `filter` → para filtrar produtos  
- ➕ `reduce` → para calcular valores (ex: total do pedido)  

---

## 🎯 Funcionalidades

- 📋 Listagem de produtos (hambúrgueres)  
- 🔎 Filtro de itens  
- ➕ Manipulação de dados com JavaScript  
- 💰 Cálculo automático de valores  
- 📱 Layout responsivo (Mobile e Desktop)  
- 🎨 Interface moderna e intuitiva
